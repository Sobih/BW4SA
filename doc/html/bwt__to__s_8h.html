char* bwt_to_s ( const struct wavelet_tree *  bwt)

A function to convert a BWT of a string back to an original string.

This function takes a BWT of a string in wavelet tree form as an input and converts it back to the original string (without $) in c-string form.

Parameters:
bwtThe bwt of a string, represented as a wavelet tree.
Returns:
The original string.
