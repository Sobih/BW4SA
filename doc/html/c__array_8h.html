Generates array C of given string in wavelet tree form.

A c-array is an array of indexes to the starting location inside a BWT of each distinct character used by the BWT, ordered lexicographically.

Parameters:
stringThe string in wavelet tree form from which a c-array should be generated.
interAn interval of the substring from which the c-array should be generated. If NULL, the entire string is used.
alphabetThe alphabet used by the string. If NULL, the alphabet of the wavelet tree is used.
alphabet_lengthLength of the alphabet supplied as parameter. If the alphabet is NULL, this parameter will be disregarded.
targetAn unsigned integer array where the results should be stored. If target is NULL, a new array will be allocated and returned instead.
Returns:
Array C which contains starting indexes of each distinct character used in the alphabet.

Topi Paavilainen, Max Sandberg (REXiator)
